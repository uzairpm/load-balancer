
# NodeJS Load Balancer

An application layer load balancer using NodeJS.





## Deployment

Start the app servers using 
```
npm run server --port 8080
npm run server --port 8081
npm run server --port 8082
```

To start the load balancer

```bash
  npm run start
```





## Roadmap
